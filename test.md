## Vue-cli 使用

##### 安装vue-cli (3.0+)
```js
    npm install @vue/cli -g 
```
##### 查看vue-cli版本
```js
    vue -V
``` 
##### 创建vue项目
```js
    vue create vue-test    / vue ui 使用ui界面创建项目
```
    选择安装 router vuex
